## 1. Executive Summary

Canton Network has no licensed, MiCA-compliant fiat on-ramp. AhoraCrypto proposes to build one: and make it freely available to every project in the Canton ecosystem.

AhoraCrypto is a Bank of Spain-registered (D803), MiCA-filing fiat on/off-ramp operating today across Europe, with licensed local payment rails active across Africa, LATAM, Asia, and MENA. We are proposing a native Canton Network integration: a production-ready, non-custodial, compliance-first fiat gateway available to any Canton application via embeddable widget iframe or REST API: with zero integration fees and zero setup costs for all Canton ecosystem partners, sustained for a minimum of three years.

This is not a speculative build. AhoraCrypto is live. The Canton integration extends existing, regulated infrastructure. Ekiden.fi: a Canton perpetual futures DEX with institutional backing: has confirmed AhoraCrypto as their sole on-ramp partner. We are filling a gap that, as of today, no other licensed provider fills on Canton.

## 2. The Gap: No Licensed On-Ramp Exists on Canton

Canton Network is maturing into institutional-grade infrastructure. Its weakest link is the fiat boundary. Today:

- **No licensed, MiCA-compliant on-ramp is natively integrated with Canton.**
- Canton DeFi applications cannot onboard retail or institutional users who need to enter from fiat: blocking TVL, trading activity, and real-world liquidity.
- Every Canton project that needs fiat on-ramp today redirects users off-network, creating drop-off, fragmented UX, and regulatory uncertainty for EU users.
- MiCA creates a winner-take-most dynamic for compliant on-ramps in Europe. The window to establish the licensed fiat standard for Canton is now: before the market consolidates.

**AhoraCrypto closes this gap: as the first and currently only MiCA-pathway, EU-registered on-ramp to integrate natively with Canton Network.**

## 3. Solution and Public Good Contribution

### 3.1 What We Are Building

- Native Canton wallet integration: non-custodial settlement directly to any Canton wallet address. Funds never touch AhoraCrypto's balance sheet.
- Embeddable white-label widget iframe: any Canton project can deploy a branded on-ramp checkout in days, with no setup fee.
- REST API: institutional-grade, supporting programmatic fiat-to-crypto order flows for wallets, DeFi protocols, RWA platforms, and AI agent workflows.
- Payment rails: Visa/Mastercard (direct approval), Google Pay, Apple Pay, SEPA, and licensed local rails covering Europe, Africa, LATAM, Asia, and MENA. Three redundant processors with active failover.
- Off-ramp (Canton-to-fiat): enabling exits as well as entries, completing the liquidity loop for Canton users.

### 3.2 What Canton Projects Get: For Free

The public good offer is concrete and durable:

- **Zero setup fees** for all Canton ecosystem projects: permanent.
- **Zero integration fees**: AhoraCrypto absorbs the cost of onboarding any Canton partner.
- **Instant revenue share**: partners earn their fee share after every transaction, with no minimums, no monthly thresholds, and no delays.
- **Re-KYC-free user experience**: AhoraCrypto and Canton ecosystem partners can cross-recognize KYC/KYB, meaning existing verified users are *not* re-KYC'd at the on-ramp step.

The banking spread and payment processing costs are passed transparently to the end user: this is how all on-ramps work. What AhoraCrypto eliminates is every discretionary fee that could otherwise be charged to Canton project teams.

### 3.3 Open Documentation and Reference Implementation

- Full REST API documentation published under open license: any Canton builder can implement fiat settlement without starting from scratch.
- Reference implementation for Canton fiat-to-crypto settlement flows: reusable by ecosystem teams.
- Compliance documentation (MiCA, KYC, AML, Travel Rule) made available to Canton participants navigating EU regulatory requirements.

Note on open source: AhoraCrypto's core backend cannot be open sourced: Visa/Mastercard approvals, acquirer relationships, and compliance frameworks require confidentiality. The public good is delivered through free ecosystem access, open documentation, shared compliance reference materials, and durable integration support.
